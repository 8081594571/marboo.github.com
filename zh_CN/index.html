Marboo(码簿笔记) Mac客户端

Download on the Mac App StoreRequires OS X 10.7 or later

自由选择编辑器

支持 Mou / Vim / Emacs / Atom / Sublime Text 等编辑器。

View details »

支持Markdown等标记语言

内置支持Markdown、reStructuredText。通过安装扩展包可以支持Org-mode等任意标记语言。

View details »

轻松浏览笔记

三栏式界面浏览笔记，宛若使用Reeder来阅读RSS文章一样。

View details »

安全，本地存储，安全无虞

笔记等文件存储在本地文件(夹)中，离开码簿笔记，仍然可以继续访问您的所有文件。码簿笔记只是实时响应您对文件(夹)的操作。

随心同步

支持使用Dropbox或Git来同步文件到不同的PC上。

笔记外观，你来决定

通过修改CSS文件来定制所有笔记的外观

局域网内，共享笔记

把笔记的链接地址告诉你的小伙伴们，同一网段的伙伴就可以点击查看。(想要关闭共享？重启码簿笔记即可)

笔记、博客，合而为一

用码簿笔记来管理Jekyll / Octopress / Hexo / Ghost 的博文是再好不过了。笔记？博客？有必要分那么清吗？发布到WordPress / Ghost 也是极好的。

代码管理，精彩呈现

内置Pygments，所有的代码文件在码簿笔记里都会语法高亮显示。
